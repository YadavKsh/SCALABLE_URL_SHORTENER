# 🔗 Scalable URL Shortener

A production-oriented URL shortener built using **Java Spring Boot**.

## 🚀 Features

* Shorten long URLs
* Redirect to original URL
* Collision handling
* Redis caching
* Scalable architecture

## 🏗️ Tech Stack

* Java + Spring Boot
* PostgreSQL
* Redis
* JMeter (for load testing)

## 📌 System Design

* Base62 encoding for short URLs
* Cache layer using Redis
* Horizontally scalable services
